#ProdeskITMission5
A Kanban Task Board built with React and Vite, inspired by Trello. It supports adding, editing, deleting, and moving tasks across To Do, In Progress, and Done columns. Includes priority-based styling (High, Medium, Low) and localStorage persistence, ensuring tasks remain saved even after refreshing the page.
